Welcome to oneDir!

Getting Started

  1. First, you need to create a oneDir account. To do this, just go to this page.

  2. Next, you need to download the oneDir client and install it locally. You can download the oneDir client from this page. To install it, run python install-client.sh from the command line.

  3. You may need to manually set the server's IP address in Watchdog/IP.txt.

  4. You're good to go! You may start the oneDir client from the command line using: python client.py
